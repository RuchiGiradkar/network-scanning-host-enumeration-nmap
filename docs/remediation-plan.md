# Remediation Plan

## High Priority

### Restrict Administrative Access

* Disable WAN administration
* Limit management access to trusted hosts
* Use strong passwords

### Firmware Updates

* Apply latest vendor firmware
* Review security advisories

---

## Medium Priority

### Denial-of-Service Protection

* Enable Anti-DoS controls
* Configure session timeout values
* Apply rate limiting

### Exposure Reduction

* Disable unnecessary services
* Remove legacy protocols

---

## Continuous Improvement

### Monitoring

* Perform quarterly network scans
* Maintain asset inventory
* Investigate unknown hosts

### Validation

* Re-run scans after remediation
* Compare results against baseline findings
